# **App Name**: Bella's Booking Bliss

## Core Features:

- User Authentication: Secure user registration and login with profile management.
- Facility Management: Admin interface to add, update, and manage resort facilities with images and pricing.
- Booking System: Allow guests to book facilities with date ranges, guest counts, and special requests.
- Admin Dashboard: Comprehensive analytics including revenue, occupancy rates, and booking statistics.
- Profile Management: Users can update personal information and upload profile pictures.
- QR Code Generation: Automatic QR code generation for guest profiles to facilitate check-in.
- AI-Powered Availability Tool: An AI tool will evaluate booking patterns to project and optimize facility availability and suggest optimal pricing.

## Style Guidelines:

- Primary color: Rausch (#FF385C) for vibrancy and Philippine resort vibe.
- Background color: White (#FFFFFF) for clean and modern aesthetics.
- Accent color: Black (#000000) for contrast and emphasis on key UI elements.
- Headings: 'Playfair Display' (serif) for elegant headlines. Note: currently only Google Fonts are supported.
- Body: 'Inter' (sans-serif) for clear and readable text. Note: currently only Google Fonts are supported.
- Use simple, outlined icons to represent different facilities and booking options.
- Mobile-first, responsive design to ensure optimal viewing experience across devices.
- Subtle animations on button hovers and transitions to enhance interactivity.
